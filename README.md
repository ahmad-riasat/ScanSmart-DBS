# ScanSmart-DBS
A simple command-line based Inventory Management System built using Python, MySQL, and Modular Functions.
The system performs CRUD operations on products, manufacture details, and billing records.

->Features
✔ Product Management

Add New Product

Show Product Details

Update Product

Delete Product

Check if a product exists

View Manufacture Details linked to a product

✔ Billing Management

View Bill History

View Bill Detail

✔ User-Friendly Menu

Continues running in a while loop

Navigation using Python match-case statement

Accepts Product ID once if needed for multiple operations

 ->Technologies Used

Python 3.10+

MySQL 

mysql.connector for database connection
